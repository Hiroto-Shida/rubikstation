# RubikStation

https://rubikstation.com/

## 機能

- トップページ
  - 図解付きのスクランブル表示
  - タイマー機能/記録
- 6 面揃えるまでの手順書

## 変更履歴 Changelog

### [2.1.1] - 2024-06-20

- 回転記号表現の矢印のスタイルを見やすく変更
